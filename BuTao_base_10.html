豐胸
  • 成因與症狀
  • 發育不良-個人本身營養的問題
  • 與個人的氣血有關
  • 對治與養生
  • 與兩個經脈有關係
  • 足陽明胃經-腸胃功能要正常-攝取營養成分
  • 足厥陰肝經-大敦穴-環繞陰器-乳頭
  • 補肝血
  • 逍遙散和加味逍遙散
  • 健運脾胃
  • 四君子湯、五味異功散、六君子湯、七味白朮散
  • 不同的藥物作用在不同的系統
  • 引經藥
  • 太陽經頭痛就用羌活
  • 陽明經頭痛就用白芷
  • 少陽經頭痛就用柴胡
  • 太陰經頭痛就用蒼朮
  • 少陰經頭痛要用到細辛
  • 厥陰經頭痛要用到吳茱萸
  • 葛根
  • 就能作用在足陽明胃經
  • 對豐胸就有很好的效果
  • 雞血藤、阿膠這一類補血的藥
  • 作息-11點到１點是骨髓造血的時間
  • 民間流傳的處方
  • 花生米燉豬皮，或是溪蝦燉酒或煮酒
  • 青木瓜燉排骨，清炒刨絲青木瓜
  • 青木瓜-帕金森症也有很好的作用


    • 縮胸
  • 成因與症狀
  • 不退奶就會脹-容易引起乳腺發炎-發熱畏寒
  • 很多食物內含荷爾蒙成分-造成胸部過大
  • 對治與養生
  • 民間流傳的處方
  • 炒大麥芽退奶，縮小胸部
  • 多吃韭菜就會消腎氣，讓乳房扁掉
  • 神麴
  • 消腎氣-過大的胸部慢慢縮小
  • 幫助消化
  • 越鞠丸-治六鬱
  • 氣鬱用香附
  • 血鬱用川芎
  • 火鬱用梔子
  • 濕鬱用蒼朮
  • 食鬱則用神麴-食鬱-指消化不良-神麴幫助消化
  • 痰鬱-丹溪先生不提供什麼藥


    • 乳房一大一小
  • 成因與症狀
  • 左血右氣
  • 左邊的問題有可能是-血的不足-過多-血的阻礙
  • 血的阻礙-活血化瘀的藥
  • 血過多-實者要瀉之
  • 血不夠-虛者補之
  • 右邊的問題有可能是-氣太多-氣不足-氣的障礙
  • 氣的阻礙-行氣的藥-破氣的藥
  • 氣太多-瀉法
  • 氣不足-補氣
  • 對治與養生
  • 柴胡桂枝湯加丹參、竹茹、桑枝、薑黃都有效
  • 針對乳房，蒲公英是必用藥
  • 左血不足用補血藥-右氣不足用補氣藥-加葛根
  • 乳頭屬足厥陰肝經-有時也加一些補肝血的藥
  • 理氣
  • 人參、黨參就能補氣
  • 陳皮、枳殼、枳實能行氣
  • 香附、木香、霍香、烏藥能行氣破氣
  • 血液
  • 當歸補血
  • 川芎行血
  • 牡丹皮、丹參能化瘀改善循環障礙
  • 左邊不夠豐滿比較小-加味逍遙散做基礎
  • 右邊比較小-四五六七系列或補中益氣湯


    • 水腫
  • 成因與症狀
  • 與腎臟有絕對關係
  • 與心肺功能有關
  • 營養不良的脾性水腫
  • 類固醇藥物會影響到腎臟功能
  • 生理週期造成大量出血-可能導致出現全身浮腫
  • 對治與養生
  • 藥物引起
  • 小柴胡湯-和解作用
  • 加上連翹、金銀花
  • 或蘆葦根、紫蘇之類-專門解魚蝦毒
  • 再加一點利尿藥物-白茅根、冬瓜子與金錢草
  • 心臟血管或心肺功能
  • 類似生脈飲的處方
  • 下肢水腫就加懷牛膝、車前子、丹參、薏仁，也可以用金錢草與白茅根
  • 脾性水腫
  • 四君子湯、五味異功散、六君子湯、七味白朮散、參苓白朮散，包括歸脾湯
  • 中滿分消湯、中滿分消丸
  • 腎臟引起的水腫
  • 腎氣丸或濟生腎氣丸
  • 再加金錢草、萆薢、白茅根、澤蘭-利水作用
  • 生理週期
  • 貧血造成的水腫-大補氣血、大補肝血
  • 加味逍遙散、人參養榮湯
  • 雞血藤、阿膠、旱蓮草、丹參、香附等來補充血液與促進骨髓的造血機能
  • 不需用消水腫的藥物，水腫也會因此豁然而癒


    • 身型不佳
  • 成因與症狀
  • 和遺傳有關
  • 臀部日益坐大
  • 腫瘤造成的大腿、手臂腫大
  • 對治與養生
  • 下肢腫大的現象-當歸拈痛湯
  • 腫瘤造成的大腿、手臂腫大-黃耆五物湯與加味逍遙散，蒲公英
  • 對任何腫瘤
  • 仙方活命飲或稱真人活命飲
  • 天花粉、浙貝母的散結
  • 玄參、鱉甲的軟堅
  • 遠志消積聚(即腫瘤)
  • 鬱金、香附、神麴
  • 荸薺


    • 增重
  • 體重的輕重是取決於我們人類的脾胃系統
  • 五臟
  • 肺主宰著皮膚毛細孔，表現在鼻腔
  • 心掌管血液與心臟血管，而表現在舌頭
  • 脾胃開竅於口腔和嘴唇，主管肌肉組織
  • 肝主管筋，而表現在眼睛
  • 腎是主骨，開竅於耳朵
  • 成因與症狀
  • 遺傳
  • 壓力
  • 產後憂鬱症
  • 更年期
  • 對治與養生
  • 健脾補氣-四五六七系列
  • 腸胃吸收-山藥、薏仁、神麴、雞內金等
  • 食道逆流、曖氣、呃酸
  • 四逆散、平胃散
  • 加一點蘆葦根與浙貝母、海螵蛸
  • 潰瘍或穿孔的現象
  • 加一點修護的藥物
  • 蔥、蒜
  • 白芨
  • 石斛-養胃聖藥
  • 嘔吐吞酸
  • 肝氣鬱結造成的
  • 四逆散-柴胡、甘草、芍藥、枳實
  • 左金丸-吳茱萸和黃連
  • 黃連-大苦大寒-少量用就是一個苦味的健胃劑
  • 吳茱萸-大熱-是一味很好的止痛劑
  • 大熱的吳茱萸和大寒的黃連
  • 壓力
  • 舒緩緊張的情緒-逍遙散、加味逍遙散、鉤藤鉤
  • 健運脾胃的方劑藥物
  • 懷孕嘔吐而消瘦
  • 小柴胡湯、香砂六君子湯
  • 加蘆葦根、桑寄生這些安胎止嘔的藥物
  • 如果還是無效-改用吳茱萸湯
  • 半夏乾薑人參丸-半夏乾薑或半夏生薑
  • 產後憂鬱症造成
  • 逍遙散、加味逍遙散-清肝、理脾、解鬱
  • 安神的藥-百合、遠志、竹茹、柏子仁等來改善睡眠與精神狀況


    • 減重
  • 成因與症狀
  • 遺傳
  • 生理周期
  • 壓力
  • 對治與養生
  • 治療胖型
  • 用化痰的藥-二陳湯、溫膽湯等
  • 活血化瘀的藥物-佛手散
  • 生理週期不規則-調經
  • 當歸芍藥散、桂枝茯苓丸、加味逍遙散、溫經湯
  • 或四物湯加桃仁、紅花
  • 遠志
  • 遠志入心-就是腦下垂體
  • 尿液不通-加白通草，通知腦下垂體做工
  • 抑制這種強烈的食慾
  • 大承氣、小承氣湯、調胃承氣湯
  • 甘露飲、玉竹、車前子、蒟蒻、黃精等
  • 孕婦如何控制體重
  • 枳朮丸或枳朮湯消減體重
  • 二陳湯、佛手散
  • 產後肥胖
  • 佛手散、溫膽湯、二陳湯、當歸芍藥散
  • 加一點車前子、金錢草、冬瓜子、白芥子這些有利水除痰的藥物
  • 更年期
  • 山楂、決明子、陳皮
  • 山楂是薔薇科植物，對消除脂肪有很好效果
  • 決明子-有便祕傾向的人-不用炒過
  • 如果大便稀軟或甚至腹瀉的人-決明子稍微炒一炒
  • 加一點車前子、金錢草、冬瓜子、白芥子這些有利水除痰的藥物
