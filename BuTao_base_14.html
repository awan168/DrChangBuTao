鼻頭粉剌、酒糟鼻
  • 定義
  • 鼻頭就是明堂，或稱準頭
  • 明堂處皮下血管充血，明顯的呈現紅紅的顏色，就叫酒糟鼻
  • 在鼻頭的地方長的痘痘可稱為粉刺，或叫痤瘡
  • 成因與症狀
  • 鼻子的病變與陽明經息息相關
  • 手陽明大腸經-商陽穴-迎香穴
  • 足陽明胃經-鼻子兩側下行
  • 飲食
  • 便祕
  • 對治與養生
  • 葛根湯
  • 竹葉石膏湯
  • 充血現象，就表示上焦有熱
  • 竹葉可以清上焦熱
  • 三焦
  • 體位
  • 功能區分
  • 譬喻
  • 經絡學-手少陽三焦
  • 加上連翹、元參、牡丹皮、桑白皮
  • 桑白皮-瀉肺熱
  • 牡丹皮-瀉血中伏火
  • 元參-瀉腎火
  • 連翹-沉澱在皮下的廢物充分代謝
  • 還可以加為諸藥之舟楫的桔梗


    • 鼻炎、鼻竇炎、過敏性鼻炎
  • 定義
  • 鼻腔黏膜-白色分泌物-黃濃稠黏-鼻炎
  • 鼻竇在鼻腔的腔洞裡-發炎-黃濃稠黏-腥臭味
  • 在鼻頭的地方長的痘痘可稱為粉刺，或叫痤瘡
  • 成因與症狀
  • 生活環境-空氣污染
  • 感冒
  • 飲食
  • 對治與養生
  • 鼻子過敏
  • 桂枝系列
  • 桂枝湯
  • 葛根湯-桂枝湯加麻黃、葛根
  • 麻桂各半湯、桂二麻一湯、桂二越婢一湯
  • 大青龍湯、小青龍湯、當歸四逆湯
  • 熱症用寒藥
  • 熱症偏向實證的情況比較多
  • 麻杏甘石湯、大青龍湯瀉白散等
  • 寒症用熱藥
  • 寒症偏虛的症狀機率比較多
  • 桂枝湯、葛根湯、小青龍湯、黃耆五物湯，當歸四逆湯等
  • 用一些荊芥、防風、桔梗、魚腥草、蟬蛻
  • 培土生金-健運脾胃
  • 四君子湯、五味異功散、六君子湯、七味白朮散、參苓白朮散、香砂六君子湯
  • 加薏苡仁-鼻病與濕、熱有關，薏苡仁有利濕的作用
  • 鼻瘜肉
  • 魚腥草-乾品洗淨，搓成棉花團，塞在鼻孔裡
  • 胃或大腸有瘜肉
  • 四逆散、樂適舒或四五六七系列的處方
  • 加魚腥草
  • 穴道和大腸經有關
  • 合谷穴
  • 迎香穴


    • 唇白、唇紅絳、唇黑
  • 定義
  • 上下眼皮，上下嘴唇-脾胃所管
  • 脾胃開竅於口唇
  • 口唇乾燥、口唇蒼白或暗黑或龜裂，一定和腸胃系統有關
  • 成因與症狀
  • 唇白
  • 嘴唇呈現蒼白的現象-判定有貧血
  • 為什麼會貧血？與腸胃消化吸收功能有關
  • 嘴唇紅絳
  • 脾熱症
  • 龜裂
  • 脾熱症
  • 氣候乾燥的地方
  • 秋天的相對濕度降低，冬天氣候冷
  • 嘴唇黑
  • 要考慮有沒有腎臟病變、腎病症候群
  • 黑
  • 青赤黃白黑對應著肝心睥肺腎
  • 黑是入腎，與腎有絕對的關係
  • 腎功能有病變，皮膚就會出現黧黑的色澤
  • 腎病症候群的人都會出現嚴重貧血的現象
  • 腎屬水，脾屬土，土能剋水，水能侮土，所以腎病也會影響到脾胃消化系統
  • 飲食習慣-天天吃冰品-口唇的顏色就變黑
  • 對治與養生
  • 唇四白呈蒼白
  • 健運脾胃
  • 四君子湯、五味異功散、六君子湯、七味白朮散、參苓白朮散、香砂六君子湯
  • 加一些補血藥
  • 嘴唇紅絳
  • 升麻葛根湯-芍藥、甘草能瀉脾熱
  • 若是因血糖升高所致，則以甘露飲治之
  • 唇黑
  • 若是因腎臟功能病變所致
  • 可以考慮用腎氣丸、豬苓湯或當歸芍藥散


    • 口腔炎、口角炎
  • 定義
  • 脾胃開竅於口唇
  • 包括口腔是指口內，口腔外就是指嘴角
  • 成因與症狀
  • 口角炎、口角潰爛-脾胃的熱症
  • 睡眠障礙、睡眠不足、失眠-一口乾舌燥-口角潰爛-口腔內炎
  • 食物
  • 荔枝、龍眼等燥熱性水果
  • 燒餅、油條、烤麵包、薯條等燥熱食物
  • 排泄不正常
  • 對治與養生
  • 睡眠障礙
  • 解鬱安神
  • 百合、柏子仁
  • 或鬱金、香附
  • 雞血藤、阿膠、旱蓮草等
  • 大便不通
  • 承氣湯類
  • 增液湯類-元參、麥冬、生地黃-滋陰養陰
  • 飲食方面-改變


    • 齒不白
  • 定義
  • 牙齒與腎臟有關
  • 齒為骨之餘
  • 成因與症狀
  • 抽煙，嚼檳榔
  • 長期營養不良
  • 刷牙
  • 對治與養生
  • 戒菸-桑白皮、蒼耳子、魚腥草
  • 營養不良
  • 上牙屬足陽明胃經，下牙屬手陽明大腸經
  • 腸胃藥如甘露飲加白芷之類的藥


    • 口臭、牙齦炎
  • 定義
  • 上下牙齦分屬胃和大腸
  • 口臭與牙周病，與腸胃系統有關
  • 成因與症狀
  • 口臭
  • 胃不和則口臭-胃不和就是腸胃不舒服
  • 胃和則口淡-胃和腸胃功能正常，口淡就是沒有異味
  • 牙周病
  • 早上起來刷牙就會流血
  • 平常口中會有特殊的味道
  • 對治與養生
  • 牙周病
  • 甘露飲-滋陰、降火
  • 加石斛、連翹、遠志、竹茹、元參等
  • 外刷骨碎補
  • 補腎強骨、續筋止痛
  • 浸酒外擦可治禿髮
  • 牙齦發炎
  • 清胃散
  • 甘露飲


    • 鼻涕倒流
  • 定義
  • 肺開竅在鼻
  • 心肺有病，鼻為之不利
  • 成方
  • 苓桂朮甘湯
  • 葛根湯
  • 桂枝有強心、擴張血管的作用
  • 葛根、芍藥有鬆弛作用
  • 可以加通竅的藥，遠志、菖蒲等
  • 小青龍湯
  • 限制：鼻涕、痰一定要是稀稀白白呈泡沫狀
  • 麻、桂、乾薑、細辛、半夏都是熱性的藥


    • 流鼻血
  • 定義
  • 熱傷陽絡則吐衂
  • 肚臍以上稱陽
  • 肚臍上的陽絡，受到熱的影響
  • 導致微細血管破裂
  • 在鼻子出來的叫做鼻衂
  • 口腔出來的叫吐血
  • 眼睛出血叫目衂
  • 耳朵出血叫耳衂
  • 熱傷陰絡則便血
  • 成方
  • 清燥救肺湯
  • 加仙鶴草
  • 也可以加元參、地黃-屬性涼，讓血管收縮


    • 打鼾
  • 原因
  • 肥胖者體型大呼吸重，容易打鼾
  • 長鼻息肉或鼻塞，以致沒辦法呼吸
  • 因感冒引起鼻塞，人難免也會打鼾
  • 成方
  • 過胖的人-減肥
  • 感冒引起
  • 麻杏甘石湯
  • 瀉白散
  • 調和營衛氣血
  • 葛根湯、大小青龍湯、苓桂朮甘湯、柴胡桂枝湯等
  • 加上芳香通竅的白芷、細辛、遠志、蒼耳子等
  • 因為鼻夾骨太窄、鼻息肉堵塞
  • 根治還是要採取外科切除
