氣胸
  • 原因
  • 感冒最容易引起氣胸
  • 心臟如果有病變也會引起
  • 對太陽膀胱經的俞穴針灸時，容易引起氣胸
  • 處方
  • 小柴胡湯、苓桂朮甘湯、枳殼、桔梗、鬱金、丹參
  • 小柴胡湯
  • 胸脇苦滿-胸腔心肺功能問題，導致呼吸困難與不利而造成急促
  • 能疏通三焦，促進人體淋巴組織回流
  • 苓桂朮甘湯-治療心肺問題
  • 桔梗、枳殼開胸利膈
  • 鬱金強心可開鬱、解鬱
  • 丹參可以強心並且活血化瘀
  • 腓腸肌骨折
  • 將鬱金、丹參換當歸、川芎
  • 當歸川芎稱佛手散，也稱開骨散-促進血液循環
  • 這裡用佛手散是治療腓腸肌骨折
  • 肺水泡引發的氣喘仍發作
  • 去當歸、川芎，加魚腥草、冬瓜子
  • 冬瓜子既可消炎，又排膿、利水


    • 症
  • 實症、熱症-麻杏甘石湯
  • 虛症、寒症-小青龍湯
  • 肺腺癌、肺腫瘤-清燥救肺湯


    • 蓮藕汁修補身體
  • 有修補有清除廢物的作用
  • 人類血管的清道夫
  • 平常多喝蓮藕汁，對肺功能、肺組織都會有很大的幫助
  • 川貝母調養肺部、百合有修補作用


    • 緩解氣喘先辨熱寒
  • 就是哮喘
  • 哮有聲音，所以以聲音命名
  • 喘則是以呼吸次數命名
  • 哮症指當情緒太過興奮或緊張時就會發生聲音
  • 哮、喘通常會同時出現，且有冷熱、虛實之分
  • 熱寒
  • 熱喘
  • 呼吸急促，有痰，鼻涕呈黃濃黏稠狀，胸口悶，呼吸提不上來，大便乾硬，且比較燥，嘴唇泛紅等
  • 用涼性藥物如麻杏甘石湯
  • 寒喘
  • 嘴唇蒼白，痰和鼻涕稀白呈泡沫狀，像蛋白一樣，大便稀稀溏溏的、手腳冰冷、呼吸急促、胸口悶等
  • 用溫熱性藥物如小青龍湯
  • 發病原因
  • 與飲食、氣候和空氣有關
  • 忌吃冰、橘子和蘿蔔、白菜
  • 有氣喘毛病的人-多食用杏仁、貝母、百合


    • 抗煞有方
  • SARS是什麼？
  • 就是急性熱性的發燒病變，最後會併發肺炎等問題
  • 麻杏甘石湯就是專門治療這些病變
  • 可以再加牛蒡、桔梗、元(玄)參
  • 如果是治療肺炎也可以再加桑白皮、魚腥草
  • 處方
  • 甘草解毒湯-甘草、黑豆、金銀花
  • 加連翹
  • 天然抗生素
  • 抗病毒的藥
  • 再加上魚腥草-對抗細菌病毒非常好的一味藥
  • 如果有發燒就加元參，喉嚨痛加牛蒡、桔梗
  • 併發腸胃性病變-葛根黃芩黃連湯
  • 白虎青龍湯
  • 鹹橄攬醃過前是青色的，中醫稱為青龍湯
  • 白蘿蔔就叫做白虎湯
  • 還可以治因感冒引起的發燒與喉嚨痛、鼻塞、過敏
  • 引發腸胃疾病的腸病毒-葛根苓連湯
  • 引發肺炎、急性肺炎-麻杏甘石湯
  • 兼有腸胃症狀，會出現嘔吐，加蘆葦根、竹茹
  • 伴隨著肺積水，則用葶藶大棗瀉肺湯
  • 心臟積水的話，就用木防己湯


    • 肺臟疾病
  • 人體的宰相，五臟之華蓋
  • 呼出是心和肺，吸入就是肝與腎
  • 肺有治理調節的功能
  • 傷風感冒-毛細孔閉塞-發燒
  • 代謝廢物-停在皮下-就會癢
  • 代謝廢物-滲透到肌肉層-就會痠痛
  • 代謝廢物-到骨骼肌-全身骨節痠痛
  • 泡熱水澡或是喝熱茶、喝熱麵湯-就會感覺舒服
  • 肺臟一開一闔進行氣體交換稱肺循環或小循環
  • 肺主氣，肺氣有問題-用人參、黃耆
  • 泌尿系統問題出現-調節肺氣達到治療目的
  • 肺葉開竅在鼻-鼻子的毛病-要從心肺看
  • 胸悶
  • 氣提不起來
  • 呼出有問題，就要用強心補肺氣的藥
  • 生脈飲，再重用黃耆
  • 人參也可以補肺氣，要讓氣升起來就用升麻、柴胡、荷葉
  • 心臟有問題-用丹參、川七
  • 如果感覺胸悶有壓迫感
  • 要用開胸利膈的藥，桔梗可以上行
  • 氣吸不進來
  • 是肝腎的問題
  • 用腎氣丸，引氣到丹田來
  • 也可以用腎氣丸加上生脈飲合用
  • 氣管炎、支氣管
  • 共同症狀為咳嗽、有痰，或因咳嗽引起胸痛
  • 治療則須分寒證和熱證
  • 寒證
  • 分泌物呈稀、白、泡沫狀
  • 小青龍湯加減服之
  • 熱證
  • 分泌物分泌物呈濃、黃、稠狀
  • 麻杏甘石湯加減，或用清燥救肺湯、麥門冬湯
  • 加冬瓜子加魚腥草
  • 介乎清、稠之間-用苓桂朮甘湯
  • 百合、薏仁、黑木耳、白木耳、貝母
  • 燉貝母與梨-可以潤肺-可以止咳-痰要比較黃
  • 痰飲
  • 黃黃稠稠黏黏的，就叫做痰
  • 稀稀白白的呈泡沬狀的那叫做飲
  • 苓桂朮甘湯-治療鼻塞、咳嗽、鼻涕倒流
