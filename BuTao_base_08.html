黑眼圈
  • 五臟六腑之精皆上注於目
  • 成因與症狀
  • 與睡眠有關
  • 熬夜
  • 黑色素沉澱
  • 飲食
  • 冰冷的食物
  • 血管、肌肉、神經收縮痙攣
  • 廢物沒辦法正常的代謝，沉澱在上下眼胞處
  • 對治與養生
  • 與睡眠有關
  • 保持睡眠充足
  • 改善睡眠狀況
  • 肝膽
  • 主管人體的任何色素變化
  • 調整肝血的處方加活血化瘀的藥
  • 赤小豆當歸散
  • 丹參-活血化瘀
  • 荊芥-辛溫升散
  • 桔梗-引諸藥上行
  • 脾胃
  • 上下眼皮是脾胃所管
  • 健運脾胃的藥如黨參、薏仁、山藥等
  • 外用藥
  • 藁本、白芷、天門冬
  • 調開水或調蜂蜜或用苦茶油調勻


    • 眼袋
  • 眼袋-中醫的名詞稱為下眼胞
  • 上下眼胞都屬於脾胃消化系統所管轄，上下嘴唇亦是
  • 成因與症狀
  • 上下眼皮的皮膚肌肉呈現鬆弛的現象
  • 腸胃消化系統功能虛衰不振
  • 過度疲勞影響到脾胃系統-勞倦則傷脾
  • 飲食習慣
  • 對治與養生
  • 身體的任何部位出現鬆弛脫垂的現象
  • 健運脾胃
  • 健脾補氣的藥物
  • 四君子湯
  • 五味異功散
  • 六君子湯
  • 七味白朮散
  • 香砂六君子湯
  • 參苓白朮散
  • 增加局部肌肉組織彈性的藥物
  • 山藥、薏苡仁、鉤藤、葛根等
  • 收斂作用的藥物
  • 芍藥、山茱萸、葛根、五倍子等


    • 長眼屎
  • 成因與症狀
  • 壓力或飲食而導致的火氣大
  • 肝膽火旺盛就會反應在眼睛
  • 嘴巴容易苦
  • 醒來上下眼皮就黏在一起
  • 大便比較乾燥
  • 飲食也誘發-譬如吃龍眼、荔枝
  • 對治與養生
  • 眼科用藥
  • 枸杞甘平
  • 菊花性涼
  • 青葙子、茺蔚子能活血化瘀
  • 石決明有鎮定的效果，還有緩瀉作用
  • 木賊草、榖精、決明子
  • 瀉火加強清熱效果
  • 梔子瀉三焦火
  • 黃芩瀉肺火
  • 清肝膽火的處方
  • 小柴胡湯-入膽經
  • 竹葉石膏湯-上升頭面


    • 多淚
  • 成因與症狀
  • 與肝血不足有關
  • 首要原因是年齡
  • 現代的燈光、紙張、電腦等
  • 睡眠不好、熬夜、透支體力
  • 高興，憂思過度
  • 不管是淚多或無淚，都是淚囊淚腺的病理表現
  • 對治與養生
  • 如果是情緒變化
  • 鎮靜安神，穩定情緒
  • 柴胡龍骨牡蠣湯
  • 溫膽湯
  • 甘麥大棗湯
  • 柴胡桂枝湯
  • 加上一些眼科用藥
  • 如果淚溢太多
  • 用一些收澀劑
  • 山茱萸、白芍藥，皆入肝膽
  • 收斂作用的五味子
  • 年齡老化
  • 杞菊地黃丸
  • 加味逍遙散
  • 加上一些眼科用藥


    • 乾眼症
  • 完全沒有眼淚叫乾眼症
  • 成因與症狀
  • 老化現象
  • 現代的燈光、紙張、電腦等
  • 對治與養生
  • 年齡老化
  • 杞菊地黃丸
  • 加味逍遙散
  • 加上一些眼科用藥


    • 紅眼症
  • 白眼球---也就是氣輪---布滿血絲
  • 成因與症狀
  • 便祕
  • 解便時會用力，一用力氣即往上
  • 白眼球是屬於肺所管
  • 肺與大腸相表裡
  • 睡眠品質不好、耗眼力、疲倦
  • 烤炸的食物
  • 工作場所
  • 對治與養生
  • 感冒
  • 太陽病
  • 紅血絲的血脈從上往下發展
  • 睛明穴-百會穴-委中穴-至陰穴
  • 葛根湯
  • 陽明病
  • 紅血絲從下到上
  • 竹葉石膏湯
  • 少陽病
  • 向外輻射或向內發展，兩側的部分
  • 小柴胡湯
  • 搭配
  • 茺蔚子、青葙子、決明子或木賊草
  • 懷牛膝、車前子等
  • 睡眠品質不好-調整作息時間
  • 烤炸的食物-盡量避免
  • 排便造成
  • 決明子有緩瀉作用
  • 若改善不大，可考慮承氣湯類-大黃
  • 竹葉石膏湯中的石膏，有涼血的作用也可以幫助排便


    • 長針眼-麥粒腫
  • 在上、下眼皮，會有一個結節硬塊
  • 成因與症狀
  • 大部分都與傷風外感有關
  • 對治與養生
  • 部位
  • 如果長在下眼皮-竹葉石膏湯
  • 如果長在上眼皮-葛根湯
  • 長在兩側的外眼角或內眼角-小柴胡湯
  • 有膿-用桔梗、連翹之類的化膿藥
  • 擴張血管的藥-荊芥、茺蔚子、青葙子
  • 排毒的藥物-連翹
  • 動作
  • 長在左(右)邊上眼皮就用右手(左)從後腦勺繞過，再用手指掰動長針眼的地方
  • 另類
  • 菊花-清熱解毒
  • 竹葉-清熱
  • 桑葉-清暑的作用
  • 青皮鴨蛋-色青入肝，肝開竅於眼睛
  • 泡茶-決明子、菊花或加點甘草片


    • 眼濁、眼珠不黑
  • 成因與症狀
  • 生理機能
  • 腎臟-排尿混濁-眼睛出現混濁
  • 肝
  • 肝經環繞陰器-排尿和肝膽也有關係
  • 有肝病的人眼睛黃濁，尿液也會從淺黃到深黃
  • 飲食-造成食物營養的沉澱
  • 睡眠品質
  • 對治與養生
  • 生理機能
  • 腎臟
  • 豬苓湯
  • 腎氣丸
  • 濟生腎氣丸
  • 杞菊地黃丸
  • 肝
  • 實症-大柴胡湯
  • 虛症-小柴胡湯
  • 搭配茵陳五苓散
  • 加上懷牛膝、車前子、金錢草、白茅根加強利尿效果
  • 眼科的用藥-決明子、車前子
  • 飲食
  • 吃比較清淡的東西
  • 混濁物質帶出
  • 活血化瘀的藥物
  • 如丹參、荊芥、川芎、茺蔚子、青葙子等
  • 枸杞、菊花、決明子
  • 生吞黑豆
  • 清肝明目，能解毒，補肝補腎，增強免疫功能
  • 四十九粒，在早上用白開水或鹽水吞下


    • 眨眼頻繁
  • 成因與症狀
  • 與現代科技有很大的關係
  • 眨眼很有可能代表一種潛在性的不安全感
  • 需要思考而造成一直眨眼
  • 眼睛過度疲勞、過度耗眼力
  • 對治與養生
  • 葛根湯
  • 再加鉤藤、秦艽、僵蠶、蟬蛻-鬆弛作用
  • 再加點石決明、柏子仁


    • 睫毛倒插
  • 成因與症狀
  • 眼皮的肌肉組織缺乏彈性
  • 太過緊張-痙攣
  • 上下眼皮是脾胃所管
  • 過度鬆弛-加強彈性
  • 過度痙攣-放鬆
  • 對治與養生
  • 葛根湯
  • 加四君子湯，或五味異功散，或六君子湯，或七味白朮散，或參苓白朮散、香砂六君子湯
  • 再加黃耆、山藥
  • 痙攣
  • 再加鉤藤、秦艽這些有鬆弛作用


    • 斜眼
  • 成因與症狀
  • 現代電視媒體
  • 日久形成
  • 對治與養生
  • 小柴胡湯
  • 加鉤藤、秦艽、遠志、石斛、榖精子
  • 放鬆


    • 虹彩炎
  • 成因與症狀
  • 眼睛使用過度有關
  • 睡眠障礙、熬夜晚睡、睡眠品質差
  • 喜歡吃烤炸的食物等
  • 眼睛脹痛就表示眼壓升高
  • 對治與養生
  • 單純的眼壓升高
  • 苓桂朮甘湯
  • 加懷牛膝、車前子、決明子、青葙子
  • 虹彩炎
  • 小柴胡湯，合竹葉石膏湯
  • 加木賊草、青葙子、茺蔚子、穀精子、蟬蛻、桔梗
